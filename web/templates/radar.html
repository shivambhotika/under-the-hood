How to read these signals

Last activity: How recently any project using this tool was updated. Active builders is not the same as GitHub stars — it counts the people actually committing code to the tool itself, not the people watching it.

A tool with 2 active builders growing at 35% is the Spark-in-2012 pattern. A tool with 1 active builder and no recent commits is a maintenance risk. Enterprise adoption (even 1 org) is early validation that the tool survived a serious engineering review.
